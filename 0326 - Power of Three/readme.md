# Power of Three

Given an integer n, return true if it is a power of three. Otherwise, return false.

An integer n is a power of three, if there exists an integer x such that n == 3x.


**Example 1:**

Input: n = 27<br>
Output: true<br>

**Example 2:**

Input: n = 0<br>
Output: false<br>

**Example 3:**

Input: n = 9<br>
Output: true<br>

**Example 4:**

Input: n = 45<br>
Output: false<br>
